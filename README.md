# Hvordan starte Todolisten 
Hvordan starte prosjektet

Klon repoet:

git clone https://github.com/Exziz08/Todolist.git


Gå til prosjektmappen:

cd Todolist


Installer avhengigheter:

npm install


Start appen:

npm start
